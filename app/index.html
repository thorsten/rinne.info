Profil

Hallo, ich bin Thorsten.

Ich bin 38 Jahre alt, wohne in Schwabmünchen und arbeite als Senior IT Architect bei der MaibornWolff GmbH in München.

Darüber hinaus programmiere ich Open Source Web Applikationen auf Basis von PHP und MySQL. Mein Hauptprojekt ist das Open Source FAQ-System phpMyFAQ. Von Zeit zu Zeit halte ich auch Vorträge rund um das Thema PHP und HTML5 auf Konferenzen weltweit.

In meiner Freizeit bin ich gerne mit Frau und Tochter unterwegs, spiele ich Handball, treffe mich mit Freunden, koche sehr gerne und arbeite mich in das Thema Fotografie ein.

Diese Seite wurde übrigens mit Yeoman, Grunt und Bower mit Hilfe von PhpStorm entwickelt. Der Code ist öffentlich unter Github einsehbar.
